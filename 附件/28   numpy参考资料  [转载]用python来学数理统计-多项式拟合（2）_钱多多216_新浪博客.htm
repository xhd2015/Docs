青山多妩媚 标记: python，数理统计，多项式，拟合

 

仍然是很简单的内容。

当然呢，看似简单的东西往往大有用处。

多项式函数的应用非常广泛，比如快速计算三角函数，经常会用到它来求，在实验数据拟合中也经常用到，此外，有空看看冯贝叶先生博大精深的《多项式和无理数》著作，明显的大吃一惊，初等方法也可以证明混沌中的逻辑斯梯映射周期三窗口的出现参数。

借助python的numpy库可以方便地进行多项式运算，包括用polyfit()函数进行最小二乘多项式拟合。

基本的操纵如下：

  • 用一维数组来表示多项式函数的系数，a[0]是最高次系数，a[-1]是常数项；
  • 多项式数组对象可以直接进行加减乘除运算，相当于对应地多项式运算；
  • 多项式函数的根可以用roots()函数计算，而ploy()函数又可由根返求多项式；

举例-1

多项式的运算


>>> import numpy as np 


>>> a = np.array([1,-2,-3]) 


>>> p = np.poly1d(a) 


>>> r = np.roots(p) #求根 


>>> print r 


[ 3. -1.] 


>>> np.poly(r) #返代 


array([ 1., -2., -3.]) 


>>> p2 = p+[2,-1] #多项式加法 


>>> print p2 


   2 


1 x - 4 


>>> print p #注意和原式比较 


   2 


1 x - 2 x - 3

举例-2

多项式最小二乘法拟合三角函数


>>> import numpy as np 


>>> import pylab as pl 


>>> x = np.linspace(-np.pi/2,np.pi/2,1000) 


>>> y = np.sin(x)-np.cos(x) 


>>> for deg in [3,5,7]: 


 a = np.polyfit(x,y,deg) 


 error = np.abs(np.polyval(a,x)-y) 


 pl.semilogy(x,error,label="$%d-order-poly$"�g,linewidth=7/deg) 


 pl.ylim(10**(-9),10**(-1)) 


 pl.xlim(-1.5,1.5) 


 pl.legend() 


>>> pl.show()

各阶多项式拟合的误差,看看吧——阶数越高,误差越小。废话。
